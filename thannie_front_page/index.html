Een onderzoek naar Automatische Beoordeling met behulp van AI

Ontvang directe, gedetailleerde feedback op jouw opdrachten. Verbeter je leerproces met onze geavanceerde AI technologie.

10 min

Per nagekeken toets

Wat kan ons programma?

Toetsen Maken

Een hele toets in 45 seconden met vragen, punten en leerdoelen.

Inscannen

Ons systeem herkent vrijwel alle handschriften en weet waar een antwoord is gegeven op een vraag.

Nakijken

Onze AI begrijpt complexe opdrachten en beordeelt net zoals een mens.

Analyseren

Ons programma analyseert alle resultaten en geeft handige feedback aan docent en leerling.

Hoe werkt het?

1

Upload je werk

Upload eenvoudig je toets naar de website.

2

Inscannen

Ons programma herkent gemakkelijk de vragen. Daarna leest hij aan de hand van AI bijna alle handschriften.

3

Nakijken

Ons programma is in staat gesloten, open en teken vragen na te kijken met een correlatie van 97% met een mens bij een echte scheikunde toets.

4

Analyseren

Elke leerling krijgt een pdf met alle feedback, per vraag en per rubric punt. Daarnaast ook zijn score per leerdoel. Als docent kan je zien welke leerdoelen goed en slecht gaan.

Wat hebben wij geleerd?

Onderzoek

  • • Meer dan 14 AI-modellen met elkaar vergeleken in meer dan 15.000 geautomatiseerde API-requests
  • • Onderzoek naar verschillende type AI-modellen en hun toepassingen, zoals: YOLO, diffusion, FFN, LLM's, latent-space-models etc.

Enquête

  • • Enquête afgenomen bij 51 docenten met verschillende achtergronden
  • • Vragen met specifiek doel gesteld, geen dubbele vragen of verschillend interpreteerbare vragen
  • • Statistische analyse d.m.v. draaitabellen, t-tests en een correlatie matrix
  • • Analyse m.b.v. AI-modellen om open vragen te analyseren en samen te vatten
  • • Duidelijke doelstelling voor ons onderzoek geformuleerd

Praktijktest

  • • Een scheikunde toets gemaakt voor een 3e klas en die gegeven
  • • Onze zelfontwikkelde blaadjes uitgelegd en de 3e klas begreep het in één keer
  • • De feedback blaadjes aan docent gegeven om aan de leerlingen uit te delen

Onderzoek uitvoeren

  • • Voornamelijk met Python (onderzoek ongeveer 4000 regels code)
  • • GitHub voor opslag
  • • Google Sheets voor analyse
  • • Python libraries zoals: Flask, Jinja, Jedi, Imutils, NumPy, Pandas, Pydantic, PyZbar, Segno, Scikit-image, SymPy, Torch, TorchVision, UV, Ultralytics, OpenCV-Python en TQDM

Backend (veilig systeem dat de berekeningen doet)

  • • 3 Python Flask servers op GCP (Google Cloud Platform): Hoofdserver, AI/GPT server en document-maak-server
  • • Route handling en onderzoek functies omgezet naar publieke routes die bestanden kunnen ontvangen en versturen
  • • GitHub voor code opslag
  • • 3500 regels handgeschreven Python
  • • Error handling geïmplementeerd
